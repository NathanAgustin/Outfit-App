# Outfit App 👔👖👟

Outfit App is an iOS application built with SwiftUI that helps users organize their wardrobe digitally, create outfit combinations, and save their favorite looks for future use.

## Overview

Choosing an outfit can be time-consuming, especially when managing a large wardrobe. Outfit App allows users to upload photos of their clothing items, build outfits virtually, and maintain a personal digital closet directly from their phone.

## Features

### Digital Wardrobe
- Upload and store clothing items
- Organize clothing into a personal closet
- View clothing collections in one place

### Outfit Builder
- Mix and match clothing pieces
- Experiment with different outfit combinations
- Visualize outfits before wearing them

### Saved Outfits
- Save favorite outfit combinations
- Revisit previously created looks
- Maintain a collection of go-to outfits

### User-Friendly Interface
- Built entirely with SwiftUI
- Clean and intuitive navigation
- Mobile-first design

## Technologies Used

- Swift
- SwiftUI
- Xcode
- iOS Development
